# 🐤 Cross Platform Resources

> Cross Platform development resource recommendations from volunteers and members on the WWCode Mobile Slack channel - #cross-platform

## React Native:
- [Udemy: The Complete React Native Course](https://www.udemy.com/course/the-complete-react-native-course-from-zero-to-hero/?couponCode=MASH2023FREE)

## Flutter:
- [Learn Flutter - Flutter's official website that's very well done and separated by level of knowledge](https://flutter.dev/learn)
- [Official Flutter YouTube Channel - fantastic content from short videos and Widget explanations to long coding sessions("The Boring Show") that are great for deeper understanding of the platform](https://www.youtube.com/@flutterdev)
- [Official Flutter Community YouTube Channel - short and long videos of different Flutter experts](https://www.youtube.com/@FlutterCommunity)
- [Tadas Petra YouTube Tutorial Channel - explains Flutter topics in a short period of time](https://www.youtube.com/@tadaspetra)
- [DBestech YouTube Tutorial Channel - showcases complete app creation videos ideal for weekend coding session](https://www.youtube.com/@dbestech)
- [Flutter Interview Topics](https://github.com/debasmitasarkar/flutter_interview_topics)

## Kotlin Compose Multiplatform:
- [KMP-Compose Multiplatform - template for getting started with Compose Multiplatform](https://github.com/SEAbdulbasit/KMP-Compose-Template)
- [KMM Awesome - repo of libraries/tools for KMP development](https://github.com/terrakok/kmm-awesome)
- [JetBrains - Compose Multiplatform mobile application tutorials](https://github.com/JetBrains/compose-multiplatform/tree/master/tutorials)
- [JetBrains - Compose Multiplatform mobile application examples](https://github.com/JetBrains/compose-multiplatform/tree/master/examples)
